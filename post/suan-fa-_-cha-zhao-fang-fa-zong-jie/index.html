2021-02-28

（1）二分法：
注意以下要点：①判空的必要②起始点要为0以防越界，判断条件为≤
//判断下开头
if (matrix[j][0]==target) {
return true;
}
//以0开始，以length结束
int low =0;
int high = matrix[j].length-1;
//条件为小于等于
while(low<=high) {
int mid = (low+high)/2;
if (target == matrix[j][mid]) {
return true;
} else if (target<matrix[j][mid]) {
high = mid -1;
} else {
low=mid+1;
}
}
return false;
